### 1. Install Ollama
- Download and install [Ollama](https://ollama.ai) on your local machine.  
- Set up any desired LLM (e.g., `llama2`, `mistral`, etc.).  
- Run the model using:
  ```bash
  ollama run <LLM name>
  ```

### 2. Install Java & Maven
- Install Apache Maven [(maven.apache.org in Bing)](https://www.bing.com/search?q="https%3A%2F%2Fmaven.apache.org%2Fdownload.cgi").  
- Install your preferred Java IDE (e.g., IntelliJ IDEA, Eclipse, or VS Code with Java extensions).

### 3. Install React & Vite
- Install [Node.js](https://nodejs.org/) (includes npm).  
- Install [Vite.js](https://vitejs.dev/).  
- Use your preferred React IDE (e.g., VS Code, WebStorm).

### 4. Run the Spring Boot Backend
- Open the `ResumeAnalyzer` folder in your Java IDE.  
- Run the Spring Boot project:
  ```bash
  mvn spring-boot:run
  ```

### 5. Run the React Frontend
- Open the `ResumeAnalyzerFrontEnd` folder in your React IDE.  
- Start the frontend:
  ```bash
  npm install
  npm run dev
  ```

### 6. Access the Application
- Open your browser and go to:
  ```
  http://localhost:5173
  ```
- Upload your resume (Word `.doc` format).  
- Add the job description.

- **Keywords** â€“ Matching terms between JD and resume.  
- **Honest Review** â€“ LLM-generated feedback.  
- **Suggested Changes** â€“ Actionable improvements for better JD alignment.
